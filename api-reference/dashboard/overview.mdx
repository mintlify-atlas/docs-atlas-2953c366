| Method | Path | Auth Required |
|--------|------|---------------|
| GET | `/api/dashboard/info` | ADMIN only |

## GET /api/dashboard/info

Returns a snapshot of platform-wide statistics including aggregate counts and detailed object lists for users, events, categories, and reports.

**Auth:** Bearer token required. Role: **ADMIN** only.

<Warning>
  This endpoint is restricted to the **ADMIN** role. Any request from a non-admin authenticated user will receive `403 Forbidden`.
</Warning>

### Response Fields

<ResponseField name="status" type="boolean">
  Indicates whether the request was successful.
</ResponseField>

<ResponseField name="statusCode" type="integer">
  HTTP status code of the response.
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable message describing the result.
</ResponseField>

<ResponseField name="data" type="object">
  The dashboard statistics object.
  <Expandable title="Dashboard object">
    <ResponseField name="usersCount" type="integer">
      Total number of registered users on the platform.
    </ResponseField>
    <ResponseField name="eventsCount" type="integer">
      Total number of events on the platform.
    </ResponseField>
    <ResponseField name="attendancesCount" type="integer">
      Total number of attendance records across all events.
    </ResponseField>
    <ResponseField name="commentsCount" type="integer">
      Total number of comments across all events.
    </ResponseField>
    <ResponseField name="users" type="array">
      List of user summary objects.
      <Expandable title="User object">
        <ResponseField name="id" type="string">
          Unique identifier of the user.
        </ResponseField>
        <ResponseField name="fullName" type="string">
          Full name of the user.
        </ResponseField>
        <ResponseField name="email" type="string">
          Email address of the user.
        </ResponseField>
        <ResponseField name="roles" type="array of strings">
          List of roles assigned to the user (e.g. `USER`, `ADMIN`, `ORGANIZER`).
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="events" type="array">
      List of event summary objects.
      <Expandable title="Event object">
        <ResponseField name="id" type="string (UUID)">
          Unique identifier of the event.
        </ResponseField>
        <ResponseField name="title" type="string">
          Title of the event.
        </ResponseField>
        <ResponseField name="description" type="string">
          Description of the event.
        </ResponseField>
        <ResponseField name="attendancesCount" type="integer">
          Number of attendance records for this event.
        </ResponseField>
        <ResponseField name="commentsCount" type="integer">
          Number of comments on this event.
        </ResponseField>
        <ResponseField name="publicationDate" type="string (ISO 8601)">
          Timestamp when the event was published.
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="categories" type="array">
      List of category summary objects.
      <Expandable title="Category object">
        <ResponseField name="id" type="string (UUID)">
          Unique identifier of the category.
        </ResponseField>
        <ResponseField name="name" type="string">
          Name of the category.
        </ResponseField>
        <ResponseField name="description" type="string">
          Description of the category.
        </ResponseField>
        <ResponseField name="createdDate" type="string (ISO 8601)">
          Timestamp when the category was created.
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="reports" type="array">
      List of report summary objects.
      <Expandable title="Report object">
        <ResponseField name="id" type="string (UUID)">
          Unique identifier of the report.
        </ResponseField>
        <ResponseField name="reporterName" type="string">
          Display name of the user who filed the report.
        </ResponseField>
        <ResponseField name="organizerName" type="string">
          Display name of the reported organizer.
        </ResponseField>
        <ResponseField name="reason" type="string">
          Reason stated in the report.
        </ResponseField>
        <ResponseField name="reportDate" type="string (ISO 8601)">
          Timestamp when the report was filed.
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

### Example

```bash
curl -X GET "https://api.meetpoint.com/api/dashboard/info" \
  -H "Authorization: Bearer <admin-token>"
```
